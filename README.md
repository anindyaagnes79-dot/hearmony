# Aksang-Cakru31
Website mengenai kegiatan aksi angkatan Cakru 31.

Sumber belajar: https://www.youtube.com/watch?v=9S4X85J7rTA&t=11834s

Cara commit di VSCode
- Buka terminal
- Pastikan git config user VSCode sesuai dengan gmail yang sudah diundang
  ```bash
  git config --global user.name "pake namamu" 
  git config --global user.email "you@example.com"
  ```
- Pastikan sudah copy repository ke laptop kalian
  ```bash
  git clone https://github.com/anindyaagnes79-dot/hearmony
  ```
- Tiap selesai menambahkan sesuatu lakukan ini
  ```bash
  git add .
  git commit -m "komen bebas habis ngapain"
  git push
  ```
- Pastikan perubahan muncul di livehost

## Cara Cek Livehost CSS

- Ikuti langkah ini di terminal
  ```bash
  npm install tailwindcss @tailwindcss/vite
  npm run dev
  ```
- Buka link server live host

## Cara Cek BackEnd JS

Di VSCode, klik kanan folder /backend
Pilih 'Open in Integrated Terminal'
```bash
npm run server
```

### Cara Cek Routes

Di VSCode, instal *extension* Thunder Client
Ikuti opsinya (get/post) serta cek tutorial di YouTube
